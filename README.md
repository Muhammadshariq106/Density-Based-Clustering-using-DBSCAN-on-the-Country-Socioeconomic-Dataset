# Density-Based-Clustering-using-DBSCAN-on-the-Country-Socioeconomic-Dataset
This project applies the DBSCAN clustering algorithm to group countries based on socio-economic indicators. Workflow includes data loading, preprocessing, EDA, feature scaling, DBSCAN clustering, PCA visualization, outlier detection, cluster analysis, and world map visualization.
Country Clustering Analysis Using DBSCAN - README

Project Overview
This project applies the DBSCAN clustering algorithm to group countries based on socio-economic indicators.
Workflow includes data loading, preprocessing, EDA, feature scaling, DBSCAN clustering, PCA visualization,
outlier detection, cluster analysis, and world map visualization.

Dataset
- File: Country.csv
- Target Column: country

Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Plotly
